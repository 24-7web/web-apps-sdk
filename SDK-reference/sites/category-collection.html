new CategoryCollection()

This class provides a collection for working with categories.

Source:

  • src/category.js

Sample code

Request

var categories = new BCAPI.Models.CategoryCollection();
categories.fetch({
    success: function (data) {
        data.each(function (category) {
            console.log(
                "Name: " + category.get('name') + " " +
                "ID: " + category.get('id') + " " +
                "parentId: " + category.get('parentId') + " " +
                "links: " + JSON.stringify(category.get('links'))
            );
        });
    },
    error: function (data) {
        console.log('Encountered an error.')
    }
})

Methods

fetch(options) → {Promise}

This method is used to fetch records into the current collection. Depending on the given options records can be filtered, sorted and paginated. For an example of how this collections are meant to be used please read BCAPI.WebApp

Parameters:

Name Type Description
options Object

Options used to control what records are fetched from server.

Properties

Name Type Description
limit Integer

The total number of records we want to fetch from server.
skip Integer

Start record index.
where Object

A JSON object containing the conditions for filtering records on server side.
order String

An attribute name by which we order records. It can be prefixed with - if you want descending order.
