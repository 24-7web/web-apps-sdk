Class: Country

BCAPI.Models.WebApp. Country

new Country()

This class provides an array of countries available for a certain webapp. In order to use this collection you must provide a webapp name. For more information regarding how to interact with the countries of a web app read BCAPI.Models.WebApp.Country.
