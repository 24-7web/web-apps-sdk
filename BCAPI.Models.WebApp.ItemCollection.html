Class: ItemCollection

BCAPI.Models.WebApp. ItemCollection

new ItemCollection()

This class provides a collection for working with web app items. In order to use this collection you must provide a webapp name. For more information regarding how to interact with web app items read BCAPI.Models.WebApp.Item.

Source:
  • src/webapp_item.js
Example
// load items for a specified webapp (only system fields are automatically loaded).
var itemCollection = new BCAPI.Models.WebApp.ItemCollection("Sample webapp");
itemCollection.fetch({
	success: function(items) {
		// handle items (only system fields available in each item).

		items.each(function(item) {
			item.fetch({
				success: function(itemDetails) {
					// do something with item details.
				}
			});
		});
	}
});
