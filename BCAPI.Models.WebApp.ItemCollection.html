Class: ItemCollection

BCAPI.Models.WebApp. ItemCollection

new ItemCollection()

This class provides a collection for working with web app items. In order to use this collection you must provide a webapp name. For more information regarding how to interact with web app items read BCAPI.Models.WebApp.Item.
