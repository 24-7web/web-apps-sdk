all the CRM records in the site

List all the CRM records on the site:

{module_webresource resource="customers" version="v3" template="" collection="customerInfo" fields="id,firstName,middleName,lastName,email1,email2,email3,username,mobilePhone,wholesaler"}
<table>
<tbody>
   {% for item in customerInfo.items %}
	<tr>
		 <td>{{item.id}}</td>
        <td>{%if {{item.firstName}} %}{{item.firstName}} {%endif%}</td>
        <td>{%if {{item.email1.value}} %}{{item.email1.value}} {%endif%}</td>
        <td>{%if {{item.email2.value}} %}{{item.email2.value}} {%endif%}</td>
        <td>{%if {{item.mobilePhone.value}} %}{{item.mobilePhone.value}}{%else%}Number unavailable{%endif%}</td>
   </tr>
   {% endfor %}
</tbody>
</table>

This example is pretty straight forward - we get all the customers on the site and using a for loop dump them in a table structure.

the "wholesale" type customers

List all the customers marked as wholesalers:

{module_webresource resource="customers" version="v3" template="" collection="customerInfo" fields="id,firstName,middleName,lastName,email1,email2,email3,username,mobilePhone,wholesaler" where="\{'wholesaler':true\}"}
<table>
<tbody>
   {% for item in customerInfo.items %}
	<tr>
		 <td>{{item.id}}</td>
        <td>{%if {{item.firstName}} %}{{item.firstName}} {%endif%}</td>
        <td>{%if {{item.email1.value}} %}{{item.email1.value}} {%endif%}</td>
        <td>{%if {{item.email2.value}} %}{{item.email2.value}} {%endif%}</td>
        <td>{%if {{item.mobilePhone.value}} %}{{item.mobilePhone.value}}{%else%}Number unavailable{%endif%}</td>
   </tr>
   {% endfor %}
</tbody>
</table>

In this example we get the list of customers that have the wholesaler value set to true. Once we get the collection customerInfo we loop it using a for statement and draw the lines in a table with the informaiton we need.
