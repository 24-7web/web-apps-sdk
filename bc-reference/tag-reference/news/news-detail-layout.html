News detail layout

This layout is not used by any module per se.

Site visitors get to it only after clicking the links outputted by the tag {tag_subject} or {tag_subject} which can both be found on the news list layout.

Accessible from

  • Admin Console:Â Site Manager > Module Templates > News Layouts
  • SFTP & Develop Mode: /Layouts/Announcement/detail.html

Renders these tags

{tag_addtofavorites,add_Image, remove_Image}Add the news item to favorites list. Optionally customize to display your own custom image for adding and removing. Here is an example - {tag_addtofavorites,<img alt="" src="/images/add.gif" />,<img alt="" src="/images/remove.gif" />}
{tag_announcementurl_value}This tag renders the announcement URL value. For example: /announcement-prefix-url/new-announcement. If customizable URLs are not enabled, this tag will render /AnnouncementRetrieve.aspx?Id=123
{tag_body}Details of the news item (editor content)
{tag_eventfromdate}Start date of the event (if applicable)
{tag_eventtodate}End date of the event (if applicable)
{tag_expirydate}Expiry date of the news item.
{tag_lastupdatedate}Last update date of the news item.
{tag_releasedate}Release date of the news item.
{tag_subject}Subject of news item in plain text
{tag_tellafriend}Opens up a popup that contains the "Tell a friend" form.
