Module menu V2 - children.html

Is used by these modules

  • {module_menu}

Accessible from

Unlike other layouts the module_menu v2 layouts can only be accessed in the "Develop" section of the site or if you connect via sFTP at this path - /ModuleTemplates/Menu/MENU_NAME/children.html

This is the Module Template used to customize the individual menu items.


Renders these tags

{tag_menuchilditem}This will render a list of all menu items with the same parent
{tag_menuitemdescription}Item Description
{tag_menuitemtargetframe}Target Frame e.g. _blank
{tag_menuitemimageurl}Image URL
{tag_menuitemimagerolloverurl}Image URL for the "rollover" state
{tag_menuitemimageselectedurl}Image URL for the "selected" state
{tag_menuitemcssclass}CSS class name
{tag_menuitemcssrolloverclass}CSS class name for the "rollover" state
{tag_menuitemcssselectedclass}CSS class name for the "selected" state
{tag_menuitemidname} Menu item ID
{tag_menuitemlabel} Menu item label
{tag_menuitemurl} Menu item URL e.g. /contact.html
{tag_menuitemcssclass_withclass}CSS class name, with  class attribute e.g. class="itemClass"
{tag_menuitemcssrolloverclass_withclass}CSS class name  for the "rollover" state, with class attribute e.g.  class="itemClassRollover"
{tag_menuitemcssselectedclass_withclass}CSS class name  for the"selected" state, with class attribute e.g.  class="itemClassSelected"
{tag_menuitemidname_withid}Item ID, with id attribute  e.g. id="myId"
