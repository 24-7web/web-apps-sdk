Webapp edit layout

Is used by these modules

Site visitors get to it only after clicking the links outputted by the tag {tag_edit} which can be found on the list layout.

Accessible from

  • Admin Console: Site Manager > System Emails > Web App Layouts > WEBAPP_NAME
  • SFTP & Develop Mode: /Layouts/WebApps/WEBAPP_NAME/edit.html (for example /Layouts/WebApps/Car Catalogue/edit.html)

Renders these tags


{tag_addtofavorites,add_Image, remove_Image}Add the news item to favorites list. Optionally customize to display your own custom image for adding and removing. Here is an example - {tag_addtofavorites,<img alt="" src="/images/add.gif" />,<img alt="" src="/images/remove.gif" />}
{tag_classifications}List of categories that this industry content is classified under
{tag_description}Description of item (editor content)
{tag_expirydate}Expiry date of item in this format - 01-Jan-9999
{tag_lastupdatedate}Last update date of item in this format - 28-May-2014 11:07 PM
{tag_name}Name of the item
{tag_releasedate}Release date of item in this format - 08-Sep-2014
{tag_weight}Weighting of item
{tag_itemurl_withhost}Full URL of the web app item including host for example http://mysamplesite.com/car-catalogue/ford-explorer
{tag_itemurl_nolink}Relative URL of the web app item for example /car-catalogue/ford-explorer
