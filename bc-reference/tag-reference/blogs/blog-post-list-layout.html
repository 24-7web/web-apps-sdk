  • Admin Console:Â Site Manager > Module Templates > Blogs Layouts

Renders these tags

Liquid tag Legacy tag Description
{{}} {tag_blogpostauthor}Author of the post
{{}} {tag_blogpostbody}Details of the post
{{}} {tag_blogpostbodypreview,Text}Display a preview of the first paragraph of your blog post. Replace "Text" with the relevant words to link to see the entire post.
{{}} {tag_blogpostdate}Date of the post
{{}} {tag_blogpostday}Day of the post
{{}} {tag_blogpostmonth}Month part of the date the blog post was posted, e.g. 11-Dec-14 will render DEC
{{}} {tag_blogpostmonthnumeric}Month part of the date the blog post was posted as a numeric value, e.g. 11-Dec-14 will render 12
{{}} {tag_blogposttitle}The title of the post as an anchor linked to the permalink
{{}} {tag_blogposttitle_nolink}The title of the post
{{}} {tag_blogpostyear}Year part of the date that the post was submitted, e.g. 11-Dec-14 will render as the year only, like this: 2014
{{}} {tag_commentcount}Number of comments for current post
{{}} {tag_commentlist}List of comments associated with this post
{{}} {tag_commentrank}Rank of comments associated with this post
{{}} {tag_permalink}Link for Permalink with anchor
{{}} {tag_permalinkonly}Link for Permalink without anchor
{{}} {tag_trackbackcount}Number of trackbacks for current post
{{}} {tag_trackbacklink}Link for trackback
{{}} {tag_trackbacklist}List of trackbacks associated with this post
{{}} {tag_blogpostid} Displays the blog post's ID
{{}} {tag_blogpostauthorpictureurl}Will render the picture url (relative to site root). e,g /images/profile.jpg
{{}} {tag_blogpostauthorpicture}Will render the full HTML image tag of the authors profile picture.
{{}} {tag_itemurl_nolink}URL to the item being referred, without host
{{}} {tag_itemurl_withhost}URL to the item being referred, with host
