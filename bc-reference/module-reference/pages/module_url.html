{module_url}

Description

This module you can use in your web pages and it will allows you to capture information that you have appended to a URL. Please see this article for further details

  • New named parameters dcumentation
  • Legacy module documentation

Syntax

{module_url,variable}

Parameters

  • variable -this is the name of the query parameter, see the example below

Examples

  • {module_url,question} - assuming the URL is mysite.com?question=how_are_you this module will display in plain text How_are_you
