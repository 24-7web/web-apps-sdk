{module_siteurl}

Description

Displays the site's URL.

This module also supports Liquid, take a look at this article to learn how to access its Liquid output tags and render it using your own custom logic.

  • New named parameters documentation
  • Legacy module documentation

Syntax

{module_siteurl,includeProtocol,URLencode collection="my_custom_collection_name" template=""}

Parameters

  • collection - optional, use this if you need to access the module's Liquid output tags
  • template - optional, use this if you need to suppress the original output and use Liquid output tags instead

This module can be rendered:

  • in plain text
  • using Liquid collections - here are the Liquid output tags you can use my_custom_collection_name.title

Examples

Non secure domain:

  • {module_siteUrl} - mtica4.businesscatalyst.com
  • {module_siteUrl,true,true} - http%3a%2f%2fmtica4.businesscatalyst.com
  • {module_siteUrl,true,false} - http://mtica4.businesscatalyst.com
  • {module_siteUrl,false,false} - mtica4.businesscatalyst.com
  • {module_siteUrl,false,true} - mtica4.businesscatalyst.com

Secure domain:

  • {module_siteUrl} - mtica4.worldsecuresystems.com
  • {module_siteUrl,true,true} - https%3a%2f%2fmtica4.worldsecuresystems.com
  • {module_siteUrl,true,false} - https://mtica4.worldsecuresystems.com
  • {module_siteUrl,false,false} - mtica4.worldsecuresystems.com
  • {module_siteUrl,false,true} - mtica4.worldsecuresystems.com
