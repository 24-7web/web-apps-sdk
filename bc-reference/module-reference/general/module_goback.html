{module_goback}

Description

Displays a link that takes the site visitor to the previous page he was visiting.

This module also supports Liquid, take a look at this article to learn how to access its Liquid output tags and render it using your own custom logic.

  • New named parameters documentation
  • Legacy module documentation

Syntax

{module_goback,text="My custom label for the goback link"}

Parameters

  • text - the text the link will render

This module can be rendered:

  • in plain text

Examples

  • {module_goback text="Take me back"} - outputs a hyperlink that takes the site visitor to the previous page he was visiting. The link's text is Take me back

Syntax

{module_goback}

Examples

  • {module_goback, Take me back} - outputs a hyperlink that takes the site visitor to the previous page he was visiting. The link's text is Take me back
