{module_goback}

Description

Displays a link that takes the site visitor to the previous page he was visiting.

  • New named parameters dcumentation
  • Legacy module documentation

Syntax

{module_goback,text="My custom label for the goback link"}

Parameters

  • text - the text the link will render

Examples

  • {module_goback text="Take me back"} - outputs a hyperlink that takes the site visitor to the previous page he was visiting. The link's text is Take me back

{module_goback}
