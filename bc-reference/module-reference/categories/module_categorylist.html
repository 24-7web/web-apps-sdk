{module_categorylist}

Description

Renders an option list of categories.

This module also supports Liquid, take a look at this article to learn how to access its Liquid output tags and render it using your own custom logic.

  • New named parameters documentation
  • Legacy module documentation

Syntax

{module_categorylist id="123" collection="my_custom_collection_name" template=""}

Parameters

  • id - the ID of the category
  • collection - optional, use this if you need to access the module's Liquid output tags
  • template - optional, use this if you need to suppress the original output and use Liquid output tags instead

Layouts

This module is not rendered with a layout.

Examples

When using this module always wrap it in a select tag like so:


<select>

{module_categorylist}

</select>

It will output a drop down containing the categories on your site


<select>

<option value="-1">-- All Categories --</option>

<option  value="32142">Category1</option>

<option  value="28940">---Category1A</option>

<option  value="43243">Category2</option>

<option  value="29324">---Category1</option>

</select>
