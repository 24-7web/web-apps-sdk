{module_webappscount}

Description

Count of items in the individual Web App.

  • New named parameters dcumentation
  • Legacy module documentation

Syntax

{module_webappscount id="webappId|webappName" filter="a(all)|c(classified)" itemId="categoryId|categoryName"}

Parameters

  • id - the web app name or the web app's ID
  • filter - filtering criteria for display and can be one of the following:
    • all - all items
    • classified - all the items in a category
  • itemId - the category ID or the category name

Examples

  • {module_webappscount id="56"} - assuming there are 4 webapp items in the webapp with the ID 56 this module will render "4"

Syntax

{module_webappscount,ID}
  • ID - system generated web app id (do not change)

Examples

  • {module_webappscount,56} - assuming there are 4 webapp items in the webapp with the ID 56 this module will render "4"
