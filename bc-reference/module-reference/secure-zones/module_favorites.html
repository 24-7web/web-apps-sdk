{module_favorites}

Description

Displays the list of favorites items for the customer that's logged in (must be logged in to secure zone). There is a limit of 500 items per page before pagination is applied.

Syntax

{module_favorites}

Examples

  • {module_favorites} - displays the items that have been added as "favorites" by the currently logged in user. The items (products, news items, faq items, etc) are displayed using their "List" layout.
