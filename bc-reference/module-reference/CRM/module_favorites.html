{module_favorites}

Description

Displays the list of favorites items for the customer that's logged in (must be logged in to secure zone). There is a limit of 500 items per page before pagination is applied.

  • New named parameters documentation
  • Legacy module documentation

Syntax

{module_favorites filter="item(individual)|all" itemType="7" itemId=""}

Parameters

  • filter - filtering criteria for display and can be one of the following:
    • item - individual item
    • all - all items
  • itemType -
    • 7 - news items
    • 48 - event items
    • 9 - FAQ items
    • 6 - media downloads
    • 27 - products
    • 35 - webapps
  • itemID - display a particular favorite item

Examples

  • {module_favorites itemType="35"} - displays the webapp items that have been added as "favorites" by the currently logged in user. The items (products, news items, faq items, etc) are displayed using their "List" layout.
