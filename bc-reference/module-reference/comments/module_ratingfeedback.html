{module_ratingfeedback}

Description

Displays user comments on an item.

  • New named parameters documentation
  • Legacy module documentation

Syntax

{module_ratingfeedback sortType="datereverse|date" template="/folder/template.tpl"}

Note: When using named parameters always wrap the individual parameters in double quotes.

Parameters

  • sortType - if this parameter is present the comments are ordered newest first
  • template - the path to the custom template you want this module to be rendered with

This module can be rendered using:

  • the comment layout - on this layout you can use either the legacy tags or Liquid output tags. Here is a list of all the tags you can use the Comment layout reference page
  • your own custom layout - take a look at the custom templates article for more details
  • Liquid collections - here are the Liquid output tags you can use:

Warning: if you need to access a liquid tag that has a space in its name, for example tag_large image_value use this syntax: {{cars1.items[0].["large image_value"]}} - Liquid does not allow for the name of its output tags to contain spaces. Examples

  • {module_ratingfeedback sortType="date"} - displays the comments newest first
  • {module_ratingfeedback template="/layouts/custom/comments.tpl"} - displays the comments using the comments.tpl custom template
  • {module_captchav2 collection="collection1" template=""} - in this case nothing will be displayed, however the module's Liquid tags will become available in the collection1 collection.
