{module_catalogue}

Description

Displays an individual catalogue on a web page..

This module also supports Liquid, take a look at this article to learn how to access its Liquid output tags and render it using your own custom logic.

  • New named parameters dcumentation
  • Legacy module documentation

Syntax

{module_catalogue catalogId="14886" rowCount="2" collection="my_custom_collection_name" template=""}

Parameters

  • catalogId - system generated catalogue id (do not change)
  • rowCount - will limit the number of items per row
  • targetFrame - e.g. _blank. Specify the frame you want the item to open in
  • collection - optional, use this if you need to access the module's Liquid output tags
  • template - optional, use this if you need to suppress the original output and use Liquid output tags instead

Layouts

  • Online Shop Layouts > Individual Catalog
  • This module also supports custom templates

Examples

  • {module_catalogue catalogId="591" targetFrame="_blank"} - will display a content of a single catalogue according to the Individual Catalogue Layout inside a new window
