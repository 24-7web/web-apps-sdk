{module_order}

Description

Displays Customer's Order History (must be logged in to secure zone).

This module also supports Liquid, take a look at this article to learn how to access its Liquid output tags and render it using your own custom logic.

  • New named parameters dcumentation
  • Legacy module documentation

Syntax

{module_order filter="all" workflowId="123" sortType="Alphabetical" collection="my_custom_collection_name" template=""}

Parameters

  • filter - filtering criteria for display and can be one of the following:
    • all - all items
    • workflow - all items in a particular workflow classification
  • workflowID - appears only when selecting "all items in a workflow" as a display criteria
  • sortType- system generated (do not change)
    • Default - Sorts the orders by date
    • Alphabetical - Sorts the orders alphabetically
  • collection - optional, use this if you need to access the module's Liquid output tags
  • template - optional, use this if you need to suppress the original output and use Liquid output tags instead

Layouts

  • Order list layout
  • This module also supports custom templates

Examples

  • {module_order filter="c" workflowID="10044" sortType="Default"} - will display all orders that belong to workflow with ID of 10044, sorting them by date
  • {module_order filter="a",Alphabetically} - will display all orders, alphabetically

Syntax

{module_order,filter,workflowID,SortType}

Parameters

  • filter - filtering criteria for display and can be one of the following:
    • a - all items
    • c - all items in a particular workflow classification
  • workflowID - appears only when selecting "all items in a workflow" as a display criteria
  • SortType- system generated (do not change)
    • Default - Sorts the orders by date
    • Alphabetical - Sorts the orders alphabetically

Layouts

  • Order list layout
  • This module also supports custom templates

Examples

  • {module_order,c,10044,Default} - will display all orders that belong to workflow with ID of 10044, sorting them by date
  • {module_order,a,,Alphabetically} - will display all orders, alphabetically, also displaying every product within each order
