{module_blogpostlist}

Description

Displays latest posts across a single blog/tag.

  • New named parameters dcumentation
  • Legacy module documentation

Syntax

{module_blogpostlist blogId="24" rowCount="10" tag="tag"}

Parameters

  • ID - system generated (do not change)
  • count - how many posts to display - the default is 10
  • tag - name of the single tag you want ot display the posts for. If is empty all the posts will be listed.

Layouts

  • Blog Layouts > List Layout
  • This module also supports custom templates

Examples

  • {module_blogpostlist,419,1,business} - will display latest post in the blog with ID 419 where the tag is the word business
