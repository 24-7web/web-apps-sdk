{module_blogsitepost}

Description

Displays latest posts across all blogs.

  • New named parameters dcumentation
  • Legacy module documentation

Syntax

{module_blogsitepost rowCount="4"}

Parameters

  • rowCount - limits the number of posts displayed by the module)

Layouts

  • Blog Layouts > Post Details
  • This module also supports custom templates

Examples

  • {module_blogsitepost rowCount="2"} - will display the most recent 2 posts on the site.

Syntax

{module_blogsitepost,numberOfPosts}

Parameters

  • numberOfPosts - limits the number of posts displayed by the module)

Layouts

  • Blog Layouts > Post Details
  • This module also supports custom templates

Examples

  • {module_blogsitepost,2} - will display the most recent 2 posts on the site.
