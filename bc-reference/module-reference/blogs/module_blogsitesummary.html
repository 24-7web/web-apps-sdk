{module_blogsitesummary}

Description

List of latest post names across all blogs (list format) or across the single blog.

This module also supports Liquid, take a look at this article to learn how to access its Liquid output tags and render it using your own custom logic.

  • New named parameters documentation
  • Legacy module documentation

Syntax

{module_blogsitesummary blogId="24" rowCount="6" collection="my_custom_collection_name" template=""}

Parameters

  • blogId - if you use -1, you will display the summary across the whole site. For a single blog you need to use the blog ID.
  • rowCount - limits the number of posts the module can display. Default is 5 posts.
  • collection - optional, use this if you need to access the module's Liquid output tags
  • template - optional, use this if you need to suppress the original output and use Liquid output tags instead

Layouts

  • Blog Layouts > List Layout

Examples

{module_blogsitesummary blogId="9216"} will output:
  • Some Philosophical questions Richard Dawkins 01-Jan-2000
  • Welcome to my Blog! Woody Mark 01-Jan-2000

As you can see above, the module displays the post titles followed by the name of the author and the date.

Syntax

{module_blogsitesummary,ID,numberfOfPosts}

Parameters

  • ID - if you use -1, you will display the summary across the whole site. For a single blog you need to use the blog ID.
  • numberfOfPosts - limits the number of posts the module can display. Default is 5 posts.

Layouts
