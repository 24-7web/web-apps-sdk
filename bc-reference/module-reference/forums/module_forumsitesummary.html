{module_forumsitesummary}

Description

List of latest posts across all forums.

Syntax

{module_forumsitesummary,numberOfPosts}

Parameters

  • numberOfPosts - it limits the number of posts the module will display

Layouts

  • This module does not use a specific layout, it uses an un-ordered list format, see the example below.
