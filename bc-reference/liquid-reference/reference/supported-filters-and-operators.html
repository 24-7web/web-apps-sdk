Supported filters and operators

Filters

Mathematical filters

divided_by {{500 | divided_by: 20}} is 25
minus {{500 | minus: 20}} is 480
plus {{500 | plus: 20}} is 520
times{{500 | times: 2}} is 1000

String manipulation filters

append {{"This is the first part" | append: " appending the second part"}} outputs "This is the first part appending the second part"
prepend {{" this is the second part" | append: "This is the first part"}} outputs "This is the first part this is thesecond part"
capitalize {{'Sample text' | capitalize}}outputs "Sample Text"
downcase {{'Sample text' | capitalize}}outputs "sample text"
upcase {{'Sample text' | capitalize}}outputs "SAMPLE TEXT"
remove {{'Sample text. All occurences of "text" will be removed.' | remove: "text"}}outputs 'Sample . All occurences of "" will be removed.'
replace {{'Sample text. All occurences of "text" will be replced.' | replace: "text", "string"}}outputs 'Sample string. All occurences of "string" will be replaced.'

Other filters

json Displays the data passed in JSON format. For example rendering a single webapp item using - {{webapp1.items[0] | json}} looks like: 
{
 "itemid":"4646524",
 "name":"ABA",
 "urlWithHost":"http://abcd.com/car/ABA",
 "url":"/car/ABA",
 "releaseDate":"2014-06-18",
 "expiryDate":"9999-01-01",
 "lastUpdateDate":"2014-06-18",
 "counter":"1",
 "weight":"",
 "Make":"AC model",
 "Color":"red",
 "Country":"",
 "Review":"no review just yet"
}
Do note you need to make the webapp listing available in a collection before you can access the items in JSON format.
date For this example let's use 2014-06-18. Here are the available switches:
  • %d - outputs the day of the month - 18
  • %M - outputs the month of the year - 6
  • %Y - outputs the year - 14

Operators

== equals
!= does not equal
> greater than
< less than
>= greater than or equal to
<= less than or equal to
or this or that
and this and that
contains You can use "contains" to check whether a string contains another string, for example: 
{% if {{webapp1.items[0].name}} contains 'Jaguar' %}
  This car is a Jaguar.
{% endif %}
or to check whether an array contains a value. Assuming {{cars_available_now}} is an array containing car makes: 
{% if {{cars_available_now}} contains 'Jaguar' %}
  A Jaguar is available now.
{% endif %}
