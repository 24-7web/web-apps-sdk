{% raw %}>{% endraw %} greater than
{% raw %}<{% endraw %} less than

{% raw %}contains{% endraw %} You can use "contains" to check whether a string contains another string, for example: 
{% raw %}{% if webapp1.items[0].name contains 'Jaguar' %}
  This car is a Jaguar.
{% endif %}
{% endraw %}
or to check whether an array contains a value. Assuming {% raw %}{{cars_available_now}} {% endraw %}is an array containing car makes: 
{% raw %}{% if cars_available_now contains 'Jaguar' %}
  A Jaguar is available now.
{% endif %}
{% endraw %}
