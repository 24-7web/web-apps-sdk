Welcome to the sample app

This sample application is provided as a reference for the Business Catalyst Developer Program. The app’s data structure is based on a custom Web App with the same name, and can be modified by changing that Web App. We encourage you to read the source code readily available in Develop mode, under the application folder (_System/Apps/bc-meet-the-team/). For more information on how to develop apps for Business Catalyst, please visit the Business Catalyst Developer Documentation.
