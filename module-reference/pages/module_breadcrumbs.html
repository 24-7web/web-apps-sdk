Description

Displays the breadcrumbs for pages according to the hierarchy defined in the site Site Map (Site Settings -> Site Map)

Syntax

{module_cataloguebreadcrumbs, separator}

Parameters

  • separator - the separator which will be used between the links

Examples

  • {module_cataloguebreadcrumbs, |} - assuming the page "Childpage" is a child item of "Parentpage" this module will render this HTML code:
    <a href="/parentpage.htm">Parentpage</a> " | " <a href="/childpage.htm">Childpage</a>
