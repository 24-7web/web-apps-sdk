{module_contentholder}

Description

Displays the individual content holder. This module also supports named content holders.

Syntax

{module_contentholder,ID}

or

{module_contentholder,name="contentholder_name"}

Parameters

  • ID - system generated (do not change)
  • or
  • name - is this represents the contentholder's name

Examples

  • {module_contentholder,34234} - will display the content holder with the ID 34234
  • {module_contentholder,name="Example"} - renders the Content Holder with name "Example" Where the SFTP file is Example.html
