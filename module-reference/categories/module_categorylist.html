{module_categorylist}

Description

Renders an option list of categories.

Syntax

{module_categorylist}

Layouts

This module is not rendered with a layout.

Examples

When using this module always wrap it in a select tag like so:

<select>
{module_categorylist}
</select>

It will output a drop down containing the catalogs on your site

<select>
<option value="-1">-- All Categories --</option><option value="32142">Category1</option><option value="28940">---Category1A</option><option value="43243">Category2</option><option value="29324">---Category1</option>
</select>
