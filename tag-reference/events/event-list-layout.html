Is used by these modules

  • {module_booking}

Accessible from

In the "Develop" section of the site or if you connect via sFTP you can find this layout at this path - /Layouts/Booking/list.html

Renders these tags


Most common tags used


{tag_name_nolink} Booking name (will not have a hyperlink) e.g. "This is the event name"
{tag_name} Booking name including a hyperlink. The hyperlink sends the user to the event's Detail Layout
{tag_body} Details of booking (editor content) e.g. "Something about the event goes here."
{tag_capacity} Total number of seats for booking e.g. "130"
{tag_percentfull} Percentage of booking seats already booked e.g. "40%"
{tag_date} Date of booking e.g. "24-Aug-2013"

Other tags


{tag_addtofavorites,addtoImage,removefromImage} Add booking to favorites list. Optionally customize to display your own custom image for adding and removing like so - {tag_addtofavorites,<img alt=" src="/images/plus_icon" />,<img alt=" src="/images/minus_icon" />}
{tag_bookingday} Day of Booking e.g. "24" if the booking date is 24-Aug-2013
{tag_bookingmonth} Month of Booking e.g. "Aug" if the booking date is 24-Aug-2013
{tag_bookingyear} Year of Booking e.g. "2013" if the booking date is 24-Aug-2013
{tag_button,Your Text} creates a link that enables the user to navigate to the event's Detail Layout (replace Your Text with your own text)
{tag_capacitydescription} Description for number of seats available. The possible values are: "Available", "Filling Up", "Almost Full" and "Full"
{tag_capacityempty} Number of available seats left for booking e.g. "79"
{tag_counter} Incremental count of the booking on the page e.g. "1"
{tag_day} Day of booking e.g. "Saturday" if the booking date is 24-Aug-2013
{tag_lastupdatedate} Date and time the booking was last updated e.g. 29-Jan-2013 08:57 PM
{tag_tellafriend} Tell a friend about this booking
