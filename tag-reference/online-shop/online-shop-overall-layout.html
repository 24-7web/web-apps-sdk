Online shop overall layout

Accessible from

In the "Develop" section of the site or if you connect via sFTP you can find this layout at this path - /Layouts/OnlineShop/page_content.html

Renders these tags

{tag_browsepanel}Displays the browse panel for current catalogue. (Use {tag_browsepanel, hideOnSale, hidePriceBrackets, hideSubCatalogues} to customize) Example of use: {tag_browsepanel,true,false,true}
{tag_cataloguebreadcrumbs}Catalogue breadcrumbs help customers easily navigate between related catalogues
{tag_cataloguelist}

Lists the sub-catalogues in the current catalogue (Use {tag_cataloguelist,x}where x isthe number of catalogues per row).

{tag_cataloguelist,rowLength,targetFrame,notUsed,sortType,hideEmptyMessage,list/table}

Change: {tag_cataloguelist,4} to {tag_cataloguelist,4,,,,,true} to display as LI.
{tag_description}Catalogue Description
{tag_name}Catalogue Name
{tag_nextpage}Next page link to traverse products in current catalogue (can be customized to use image)
{tag_pagination}Enhanced pagination e.g. "1 .. 2 3 4 5 .. 10" to provide fast access to adjacentpages
{tag_previouspage}Previous page link to traverse products in current catalogue (can be customized to useimage)
{tag_productlist}Lists the products in the current catalogue (Use {tag_productlist,u,v,w,x,y,z} where u is the number of products per row; v is the target frame, e.g. _blank or leave empty; w is the number of products per page (limit of 500 products per page); x is the sort type, e.g. Price, or Alphabetical; y allows you to hide empty message, e.g. 'this catalog has no products' and z is list type). Change: {tag_productlist,4} to {tag_productlist,4,,,,,true} to display as LI.
{tag_shoppingcartsummary}Shows shopping cart summary with view cart link {tag_shoppingcartsummary,horizonal} and {tag_shoppingcartsummary,vertical} to display horizontally and vertically respectively, {tag_shoppingcartsummary,,quote} to display total.
