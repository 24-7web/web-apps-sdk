Blog comment list layout

Is used by these modules:

  • This is actually called through {tag_commentspaged}, which can be found in the Detail Layout

Accessible from

In the "Develop" section of the site or if you connect via sFTP you can find this layout at this path - /Layouts/Blog/comment.html

Renders these tags

{tag_feedback}Customer comment
{tag_fullname}Full name of the person who submitted the comment (anonymous if not available)
{tag_fullname_nolink}Full name of the person who submitted the comment without a link (anonymous if not available)
{tag_rating}Rating in stars
{tag_usagedate}Date when comment was given
{tag_pendingcomments}Checks if status of the comment is set to "pending" from admin, in whic case outputs the following message:"Your comment is awaiting moderation!"
