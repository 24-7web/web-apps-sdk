Accessible from

In the "Develop" section of the site or if you connect via sFTP you can find this layout at this path - /Layouts/CustomerCases/detail.html

Renders these tags


{tag_assignedto}Customer service representatives the case is assigned to (Ex. John Smith)
{tag_casenumber}Case number
{tag_createdate}Date and time the case was first created (Ex. 25-Mar-2013 01:46 PM)
{tag_lastupdatedate}Date and time the last message was received (Ex. 25-Mar-2013 01:46 PM)
{tag_ownername}Fullname of the owner of the case (Ex. John Smith)
{tag_crmform}The data from all CRM forms attached to a case (Ex. In the case where a user submits a standard contact form, this tag will output the following code)
{tag_description}Description of the case
{tag_filelist}List of files attached to a case (can be downloaded)
{tag_status}Status of the Case (Can be: New, Open, Escalated, Closed, Re-Opened)
{tag_subject}Subject of Case
