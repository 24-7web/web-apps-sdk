Double opt-in email

Accessible from

In the "Develop" section of the site or if you connect via sFTP you can find this layout at this path - /Layouts/OutboundEmails/DoubleOptInEmail.html

Renders these tags

{tag_emailaddress}Email address of the recipient
{tag_recipientname}Full name of the recipient
{tag_verificationurl,Type}The verification url for customer to select and opt-in to your newsletter. Type is either Html or Text. Html will render <a href="verifyUrl">Your_html</a> and text mode, your_text verifyUrl
