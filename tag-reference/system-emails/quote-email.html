Quote email layout

Accessible from

  • Admin Console: Site Manager > Online Shop Layouts > Shopping Cart
  • SFTP & Develop Mode: /Layouts/OnlineShop/shopping-cart-XX.html

    Replace "XX" with the country code you have added in the Site Settings > Site Domains panel (see the "Country" column). 

    If you have multiple domains with different "Country" values you will find multiple shopping-cart-XX files, one for each country. This way, with one layout for every country defined you can vary the contents of the layout to accomodate a multilingual website.
