What is this layout for?

This layout is used for the "Tell-a-friend" email the persons referred by a site visitor receives. For more info on the "Tell a friend" feature please see the Tell a Friend description article.

Accessible from

In the "Develop" section of the site or if you connect via sFTP you can find this layout at this path - /Layouts/OutboundEmails/ReferAFriend.html
