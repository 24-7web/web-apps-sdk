Comments layout

Is used by these modules

  • {module_ratingfeedback}

Accessible from

In the "Develop" section of the site or if you connect via sFTP you can find this layout at this path - /Layouts/Comments/comments.html

Renders these tags

{tag_country}Country of the person who provided the comment
{tag_feedback}Customer comment
{tag_flag}Image of the flag of the country of the person who provided the comment
{tag_fullname}Displays the full name of the person who provided the comment.If the "Website" field was not blank when the comment form was submitted the name is rendered as an anchor. If the "Website" field was blank at the time the comment form was submitted the output is in plain text. If the comment was submitted without having the "Name" field filled in "Anonymous" is displayed.
{tag_fullname_nolink}Full name in plain text of the person who provided the comment. If the comment was submitted without having the "Name" field filled in "Anonymous" is displayed.
{tag_rating}Rating in stars
{tag_usagedate}Date when comment was given in this format - 27-Mar-2013 12:02
{tag_website}Website of the person who provided the comment. If the comment form was submitted with the "Website" field not filled in this tag will output "{tag_website}"
