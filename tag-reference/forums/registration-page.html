Forum registration layout

Is used by these modules

This layout is not used by any modules. The site visitor gets to see it if he is not logged into the site's front-end and attempts to perform an action that requires his being authenticated - create a new post, subscribe to a topic or post a reply.

Accessible from

In the "Develop" section of the site or if you connect via sFTP you can find this layout at this path - /Layouts/Forum/registration_page.html

Renders these tags

By default on this layout there is only a login form and a new user registration form. This layout does not contain any tags.
