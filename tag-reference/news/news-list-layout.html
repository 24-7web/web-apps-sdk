News list layout

Is used by these modules

  • {module_announcement}

Accessible from

  • Admin Console:Â Site Manager > Module Templates > News Layouts
  • SFTP & Develop Mode: /Layouts/Announcement/list.html

Renders these tags

{tag_addtofavorites,addtoImage,removefromImage}Add the news item to favorites list. Optionally customize to display your own custom image for adding and removing. Here is an example - {tag_addtofavorites,<img alt="" src="/images/add.gif" />,<img alt="" src="/images/remove.gif" />}
{tag_body}Details of the news item (editor content)
{tag_eventfromdate}Start date of the event in this format - 17-Sep-2014
{tag_eventtodate}End date of the event (if applicable)
{tag_button,Your Text}will output a hyperlink with the text "Your Text" and when clicked will take the site visitor to the detail layout
{tag_expirydate}Expiry date of the news item
{tag_lastupdatedate}Last update date of the news item
{tag_releasedate}Release date of the news item
{tag_subject}Subject of the news item (will have hyperlink) that when clicked will open the news item detail layout
{tag_subject_nolink}Subject of the news item in plain text.
{tag_announcementurl_value}Will output the URL of this news item, for example /announcements/important-notice-to-shareholders
{tag_counter}Outputs the current number of this news item on the page.
{tag_announcementid}Will output the ID of this news item, for example 15234
{tag_tellafriend}Opens up a popup that contains the "Tell a friend" form.
