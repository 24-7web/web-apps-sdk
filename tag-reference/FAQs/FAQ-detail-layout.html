FAQ detail layout

This layout is not used by any module per se.

Site visitors get to it only after clicking the links outputted by any of these tags {tag_button,Your Text} or {tag_question}.

Accessible from

  • Admin Console:Â Site Manager > Module Templates > FAQ Layouts
  • SFTP & Develop Mode: /Layouts/FAQ/detail.html

Renders these tags

{tag_addtofavorites,addtoImage,removefromImage}Add FAQ to favorites list. Optionally customize to display your own custom image for adding and removing. Here is an example - {tag_addtofavorites,<img alt="" src="/images/add.gif" />,<img alt="" src="/images/remove.gif" />}
{tag_answer}Answer for frequently asked question (editor content). This outputs the text of the FAQ.
{tag_classifications}List of categories that this FAQ is classified under. The categories are in plain text and are sepparated by commas for example Company, Products
{tag_expirydate}Expiry date of FAQ for example 16-Apr-2014 03:48 AM
{tag_internalnotes}Outputs the text entered in the "Internal FAQ Information" textbox.
{tag_lastupdatedate}Last update date of FAQ for example 16-Apr-2014 03:48 AM
{tag_question}Frequently asked question (will have hyperlink). This outputs the FAQ's title as an anchor. When clicked the site visitor is taken to the detail view. The text of the anchor is the FAQ's title.
{tag_releasedate}Release date of FAQ in this format - 16-Sep-2014
{tag_tellafriend}Tell a friend about this FAQ. This outputs a link that when clicks opens up a popup that contains the "Tell a Friend" form.
