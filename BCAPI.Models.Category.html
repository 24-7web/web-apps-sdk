Class: Category

BCAPI.Models. Category

new Category()

This class provides a way of working with individual category items.

Source:
  • src/category.js
Example
var category = new BCAPI.Models.Category({name: 'Test Category'});
To save:
category.save(options)
To get a category by id:
var category = new BCAPI.Models.Category({id: 1});
category.fetch(options)

Update and delete are not supported

sync(method, model, options) → {Promise}

Sync method is invoked automatically when user tries to create / update a model. It automatically appends the custom headers returned by BCAPI.Models.Model.headers method.
