Class: Model

BCAPI.Models. Model

new Model()

Model provides the base class for every model included in Business Catalyst client side sdk. Each new model inherits Model.

Source:
  • models.js, line 8
Example
BCAPI.Examples.Person = BCAPI.Models.Model.extend({
	defaults: {
		firstName: "John",
		lastName: "Doe"
	},
	endpoint: function() {
		return "/api/v2/persons";
	}
});

headers() → {Object}

This method returns the predefined headers which are automatically appended to ajax calls. For instance, Authorization header must be set to site token for most of the calls. If you need a different behavior in your model, please override this method.

sync(method, model, options) → {Promise}

Sync method is invoked automatically when user tries to create / update a model. It automatically appends the custom headers returned by BCAPI.Models.Model.headers method.
